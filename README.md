
# Charity DApp

Ứng dụng React đơn giản để kết nối ví MetaMask và donate ETH đến smart contract.

## Cách sử dụng

1. Cài Node.js (nếu chưa có): https://nodejs.org
2. Mở terminal, chạy:

```
npm install
npm start
```

3. Truy cập `http://localhost:3000`
4. Kết nối MetaMask và donate ETH
